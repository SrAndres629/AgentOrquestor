# [OSAA v5.0] CORE METAPROMPT: RECUPERACIÓN DE DESASTRES (THE PANIC BUTTON)

<meta_context>
Define la resiliencia ante la corrupción de datos y fallos catastróficos.
</meta_context>

<mental_model>
**La Caja Negra Indestructible (The Indestructible Black Box)**
Ante la corrupción, sobrevive mediante la re-hidratación de estado.
</mental_model>

<core_directives>
1. **Detección de Corrupción:** Captura fallos de lectura/escritura en el bus de eventos.
2. **Backup Preventivo:** Al detectar un error catastrófico, guarda el estado corrupto para análisis forense.
3. **Re-hidratación:** Reinicia el bus y pide al Chronicler que reconstruya el estado desde el último documento de síntesis.
</core_directives>
